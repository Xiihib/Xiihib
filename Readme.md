# ¡Hola! 👋 Soy Nahib Mubaharak

Soy **Desarrollador Junior Full Stack** con experiencia en **Python** y **Django**, e **Ingeniero en Telecomunicaciones, Conectividad y Redes**.

Mi formación en telecomunicaciones me permitió desarrollar una base sólida en **redes**, **infraestructura** y **resolución de problemas**. Con el tiempo descubrí mi interés por el desarrollo de software, donde encontré una forma de combinar la lógica, la creatividad y el aprendizaje continuo para construir soluciones que generen valor.

Actualmente me encuentro ampliando mis conocimientos en desarrollo **backend** y **frontend**, aplicando buenas prácticas de programación, bases de datos y control de versiones con **Git**. También estoy profundizando en tecnologías relacionadas con el desarrollo de aplicaciones web y la ciberseguridad.



## 🎯 Objetivos
